Title: Free IPTV Player
Short description: Смотрите ТВ, фильмы и сериалы из ваших M3U плейлистов на любом Android-устройстве.
Full description:
Free IPTV — мощный медиаплеер для Android TV, смартфонов и планшетов. Транслируйте личный IPTV-контент с современным и удобным интерфейсом, разработанным для сенсорного управления и навигации с пульта.

ВОЗМОЖНОСТИ:
• ТВ-каналы в прямом эфире с EPG (Электронный Программный Гид)
• Фильмы (VOD) с постерами из TMDB
• Сериалы с навигацией по сезонам и эпизодам
• Поддержка нескольких аудиодорожек и субтитров (встроенные + внешние через OpenSubtitles / SubDL)
• Продолжить просмотр — возобновите воспроизведение с того места, где остановились
• Избранное, оценки и история просмотра
• Режимы отображения: сетка и список
• Умный поиск и фильтры по категориям
• Поддержка прокси потоков для VPN/маршрутизации
• Удалённая настройка из любого веб-браузера
• 11 языков интерфейса: русский, английский, французский, немецкий, испанский, итальянский, португальский, нидерландский, польский, арабский, турецкий
• Оптимизирован для Android TV с навигацией D-pad

ПОДДЕРЖИВАЕМЫЕ ФОРМАТЫ:
• Плейлисты M3U / M3U8
• API Xtream Codes
• Медиасерверы Jellyfin
• Потоковое вещание HLS и DASH (на базе ExoPlayer)

КОНФИДЕНЦИАЛЬНОСТЬ:
• Без рекламы, без отслеживания, без сбора данных
• Все настройки хранятся локально на вашем устройстве
• Политика конфиденциальности: https://iptv.blanquer.org/privacy.html

ОТКАЗ ОТ ОТВЕТСТВЕННОСТИ:
Это приложение не предоставляет контент. Пользователи должны настроить собственные IPTV-плейлисты и несут полную ответственность за законность доступа к такому контенту.

Требования: подключение к Интернету, плейлист M3U или сервер Xtream Codes / Jellyfin
