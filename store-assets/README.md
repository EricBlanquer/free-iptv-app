# Store assets

Listing assets for the app stores. Uploads are **manual** on each console —
the local files are not pushed automatically.

- `screenshots/` — **shared** screenshots (Google Play + Whale TV), 1920×1080,
  Aurora theme. Single source: regenerate here to update both listings.
- `google-play/` — Google Play only: `feature-graphic.png`, `remote-config.jpg`,
  localized descriptions `*.txt`.
- `whale/` — Whale TV only: `banner-320x180.png`, `icon-300.png`.
