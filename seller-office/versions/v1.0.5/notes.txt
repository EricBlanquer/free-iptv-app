=== Note for Tester ===
No login required.

At first launch, a welcome popup appears: "Would you like to try a demo playlist adapted to your location?". Select "Yes, try demo" — the app automatically adds a geo-adapted playlist (served from https://iptv.blanquer.org/playlist.m3u, detecting the tester's country from the request IP) and opens the Live TV section. Select any channel to test playback.

Alternatively, select "No, configure manually" to reach Settings > Playlists and add your own source:
- M3U type: enter a playlist URL
- Server type: enter host URL, username, password

The app does not host any content. This version adds the welcome demo playlist, auto-jumps to Live TV when only live content is available, and fixes false "frozen" playback detections that were triggering unwanted reconnections on otherwise normal streams.
