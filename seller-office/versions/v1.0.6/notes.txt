=== Note for Tester ===
WHAT'S NEW

Fixes the v1.0.5 rejection ("No data available" on demo). The demo playlist is now bundled inside the .wgt (no external server required). Added offline detection: clicking a channel without internet shows a clear "No internet connection" message instead of hanging on the loading spinner. Material Symbols font and channel logos are now embedded (no CDN dependency). New: Page Up/Down keys to scroll by page, alphabetical sort on Live channels, mouse/touch scroll with focus tracking.

HOW TO TEST

No login or account required.

Option A — Try the bundled demo (recommended):
  1. On first launch, accept "Would you like to try a demo playlist?".
  2. App auto-jumps to Live TV with 7 channels (DW family, France 24 FR/EN, Al Jazeera English).
  3. Click any channel — should start playing within seconds.

Option B — Configure manually:
  Settings > Playlists > Add Playlist, enter an M3U URL, or Xtream credentials (server URL, username, password).
