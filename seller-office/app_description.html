1. UI Structure

Home Screen
├── Live TV → Browse (categories + channel list) → Player
├── Movies (VOD) → Browse (categories + movie grid) → Details → Player
├── Series → Browse (categories + series grid) → Details (seasons/episodes) → Player

    ├── Remote Configuration (QR code)

    ├── Text-to-Speech
    ├── Stream Proxy
    ├── APIs (TMDB, OpenSubtitles, SubDL)
    └── Premium / Support

2. Usage Scenario

First Launch (No playlist configured)

  1. App launches and shows Home screen with only "Live TV" and "Settings" buttons
  2. User navigates to Settings
  3. User selects "Manage playlists" and presses Enter
  4. User selects playlist type: "M3U" or "API Server"
  5. For M3U: user enters a playlist URL (e.g., https://iptv-org.github.io/iptv/index.m3u)
  6. For API Server: user enters server URL, username, and password
  7. User selects "Save" and returns to Home
  8. App loads playlist data and shows all available sections (Live TV, Movies, Series, etc.)

Watching Live TV

  1. From Home, user selects "Live TV"
  2. Browse screen shows categories on the left, channels on the right
  3. User navigates categories and selects a channel
  4. Player starts streaming the selected channel
  5. User can change channels with Up/Down keys
  6. Press Return to go back to channel list

Watching a Movie (VOD)

  1. From Home, user selects "Movies"
  2. Browse screen shows movie categories and movie posters
  3. User selects a movie to see details (synopsis, cast, rating from TMDB)
  4. User selects "Play" to start playback
  5. During playback: Left/Right to seek, Enter to pause/resume
  6. User can select subtitle tracks and audio tracks

Remote Configuration

  1. User opens Settings
  2. Under "Remote Configuration", a QR code is displayed
  3. User scans QR code with phone/computer
  4. Opens iptv.blanquer.org where user can configure playlists remotely
  5. Configuration is synced to the TV app

3. Menu & Function Description

3.1 Home Screen

Grid of buttons to access available sections. Live TV and Settings are always visible.
Movies, Series, Sport, Manga, Entertainment and History sections appear only when an API Server playlist is configured.

3.2 Browse Screen (Live TV)

Left panel with categories. Right panel showing channels. Navigate with Up/Down, select with Enter. Grid or list view toggle available (RED key).

3.3 Playlist Management

Add, edit, delete M3U playlists or API server connections. Supports M3U URL and Xtream API server credentials.

3.4 Settings Screen

Configuration options: playlist management, language selection, remote configuration via QR code, license status, API keys, text size, text-to-speech.

3.5 Remote Configuration

QR code displayed in Settings. User scans with phone/computer to configure playlists remotely — configuration is synced to the TV app via iptv.blanquer.org

5. How to Change Languages

ItemDetails
How to change languages Go to Settings → Language → Select desired language from the list.

Supported languages: English, French, German, Spanish, Italian, Portuguese, Dutch, Polish, Russian, Arabic, Turkish.

The app UI language changes immediately after selection. No restart required.
